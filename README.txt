MetadataDetector - CLI-программа для проверки наличия метаданных HDR10plus в файлах .mp4

Поддерживаемая платформа: x64

Содержимое корневой папки:
- папка 'metadataDetector'
- папка 'x265lib'
В папке 'metadataDetector' содежится папка 'Debug' с исполняемым файлом 'MetadataDetector.exe' и вспомогательными библиотеками
В папке 'x265lib' содержатся вспомогательные библиотеки и заголовочные файлы

Правила использования:
- принимает на вход два аргумента '-i' и '-j'
- аргумент '-i' указывает путь к видеофайлу .mp4
- аргумент '-j' указывает путь к .json файлу с метаданными

Пример использования:
В папке находятся файлы 'test_video.mp4' и 'test_video.json'
Команда для запуска программы: MetadataDetector.exe -i test_video.mp4 -j test_video.json

В этой версии исправлен баг с определением размерности файла через fseek
